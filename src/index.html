One-Minute Setup

1
Install Ollama

Download from ollama.com and launch it.

2
Download a Model

Run ollama run llama3.2 (or any model of your choice) in your terminal.

3
Launch KobiTab

Open the app and start chatting with your knowledge base.

$ ollama serve
listening on 127.0.0.1:11434...

$ ollama run llama3.2
success! Kobi is ready.
